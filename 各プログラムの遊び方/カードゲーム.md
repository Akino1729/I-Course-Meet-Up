# カードゲーム

## 全体ルール
1. 各班で班長を1人決める
2. 班長が前（景品・菓子エリア）に来る
3. 司会とじゃんけんをして、**勝った班長からやりたいカードゲームを選んで取る**
4. 各班でカードゲームを行い、**最初にあがった人**から前のお菓子を取りに行く（1人1回のみ，お菓子は2個まで）

（内訳）UNO × 18（シリーズ違い色々）、音速飯店 × 2

> **どのUNOかわからない場合は箱を確認してください。**

## UNO（オリジナル系）の遊び方

> オリジナル、マリオカード、ポケモン、ゴールドが当たった班が対象

### 準備
- よくシャッフルして、各プレイヤーに7枚ずつ配る
- 残りを山札として中央に置き、1枚めくって捨て札置き場を作る

### ゲームの流れ
1. 時計回りに順番が回る
2. 自分の番に、**捨て札と同じ色・数字・記号**のカードを出す
3. 出せない場合は山札から1枚引く。引いたカードが出せる場合はそのまま出してよい
4. 手札が残り1枚になったら**「UNO！」と宣言する**（忘れると山札から2枚引くペナルティ）
5. 手札を全部出し切ったらあがり

### 記号カード

| カード | 効果 |
|---|---|
| スキップ | 次の人の番をとばす |
| リバース | 順番の向きが逆になる |
| ドロー2 | 次の人が2枚引いて1回休み |
| ワイルド | 色を自由に指定して出せる |
| ワイルドドロー4 | 色を指定＋次の人が4枚引いて1回休み |

## UNO（オールワイルド）の遊び方

### オリジナルとの違い
- **全カードがワイルド**扱いなので、場のカードが何であっても**いつでも自由に出せる**
- 色・数字の一致は一切不要。手番が来たら好きなカードを出すだけ

### ゲームの流れ
準備・あがり条件・UNO宣言はオリジナルと同じ。カードの効果は以下のとおり。

### カード一覧

| カード | 効果 |
|---|---|
| ワイルド | 効果なし。好きなタイミングで出せる |
| ワイルドリバース | 順番の向きが逆になる |
| ワイルドスキップ | 次の人の番をとばす |
| ワイルドスキップ2 | 次の**2人**の番をとばす |
| ワイルドドロー2 | 次の人が2枚引いて1回休み |
| ワイルドドロー4 | 次の人が4枚引いて1回休み |
| ワイルド指名ドロー2 | **指名した人**が2枚引いて1回休み |
| ワイルド強制スワップ | 指名した相手と手札を**全部交換**する（交換後1枚なら「UNO！」宣言） |

## UNO（フリップ）の遊び方

### 基本
カードに**ライトサイド（表）とダークサイド（裏）**の2面がある。ライトサイドで開始し、フリップカードが出たら全員・山札・捨て札をひっくり返してダークサイドへ切り替わる。

### 準備
- 各プレイヤーに7枚ずつ配る
- ライトサイドでゲームを開始する

### ライトサイドのカード

| カード | 効果 |
|---|---|
| ドロー1 | 次の人が1枚引いて1回休み |
| リバース | 順番の向きが逆になる |
| スキップ | 次の人の番をとばす |
| フリップ | **ダークサイドへ切り替え**（全員・山札・捨て札をひっくり返す） |
| ワイルド | 色を自由に指定して出せる |
| ワイルドドロー2 | 色を指定＋次の人が2枚引いて1回休み |

### ダークサイドのカード
> ダークサイドの色：ピンク・オレンジ・水色・パープル

| カード | 効果 |
|---|---|
| ダークドロー5 | 次の人が5枚引いて1回休み |
| ダークリバース | 順番の向きが逆になる |
| ダークスキップ | **自分以外の全員**の番をとばす |
| ダークフリップ | **ライトサイドへ切り替え**（全員・山札・捨て札をひっくり返す） |
| ダークワイルド | 色を自由に指定して出せる |
| ダークカラーワイルド | 色を指定＋次の人は**その色のカードが出るまで引き続ける** |

## 音速飯店の遊び方

### 概要
中華料理の名前をカードで完成させる**早出しバトル**。自分の番はなく、常に早い者勝ちで進む。

### 準備
1. 全カード（60枚）を全員に均等に配り切る（例：8人なら7〜8枚ずつ）
2. カードはまだ手に取らない
3. 全員の準備ができたら**「ちゅうも〜ん！」**の掛け声でラウンド開始、一斉に手に取る

### ゲームの流れ
- 自分の番はない。**早い者勝ちでどんどんカードを出す**
- 中華料理の名前を**最初の文字から順番に**完成させるようにカードを出す
  - 例）ラーメンを作る場合：「ラー！」→「メン！」の順に出す
- カードを出すときは**声に出して読み上げる**（「チャー！」など）
- 同じ人が連続して出してもOK
- **とりけしカード**：料理が作られている途中にいつでも割り込める。リズムが崩れるので使いどころが重要

### ラウンドの勝ち抜け
- 手札最後の1枚を**「あがり！」**とコールしながら出せたらそのラウンド勝ち抜け
- ラウンドはすぐ終了せず、残ったプレイヤーで続ける
- **最後の1人になった人が負け抜け**でラウンド終了
- 誰も出せなくなった場合も、手札が残っている全員が負け抜けでラウンド終了

### ポイント
- 箱の内側に**お品書き（作れる料理一覧）**があるので確認しておくと有利
- カードは角に文字が書かれていないため、**手を動かして手札を把握する**必要がある
